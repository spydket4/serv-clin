TCP Эхо-сервер с автоответами
Простой TCP эхо-сервер на Python, который отправляет эхо-ответы на сообщения клиента.

Особенности
- Сервер отправляет эхо-ответы с нумерацией сообщений
- Клиент может в любой момент отправить "exit" для завершения соединения
- Реализована обработка разрыва соединения и таймаутов
- Кросс-платформенное решение (работает на Windows, Linux, macOS)

Структура проекта
tcp-echo-server/
├── server.py          # Серверная часть
├── client.py          # Клиентская часть
└── README.txt         # Этот файл

Использование

Запуск сервера:
1. Откройте терминал и перейдите в директорию проекта
2. Запустите сервер: python server.py
3. Сервер по умолчанию слушает на:
   - Хост: 0.0.0.0 (все интерфейсы)
   - Порт: 13337

Запуск клиента:
1. Откройте другой терминал
2. Измените IP-адрес в client.py на IP-адрес вашего сервера
3. Запустите клиент: python client.py

Важно: В файле client.py измените IP-адрес на IP-адрес вашего сервера:
HOST = "192.168.1.100"  # Замените на реальный IP сервера

Настройка сервера (server.py):
HOST = "0.0.0.0"  # Слушать на всех интерфейсах
PORT = 13337       # Порт для подключения

Настройка клиента (client.py):
HOST = "127.0.0.1"  # localhost для тестирования на одном компьютере
PORT = 13337        # Должен совпадать с портом сервера
